
# Notes API (Node.js)

This is a backend Notes API built using Node.js, Express, and PostgreSQL.

## Features
- User authentication (JWT)
- Create, update, delete notes
- PostgreSQL database integration

## Tech Stack
- Node.js
- Express.js
- PostgreSQL
- JWT Authentication

## Setup

```bash
npm install
node server.js
